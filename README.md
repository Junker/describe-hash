# Describe Hash

## Intoduction

describe-hash provides help function for examining a hash map

## Instalation
It's available on Melpa:
```
M-x package-install describe-hash
```

## Usage
``` emacs-lisp
(require 'describe-hash)
(global-set-key (kbd "<f1> C-h") #'describe-hash)
```
